WaterMe - a #RHoKSoton project

This repository contains the code for the WaterMe project developed at the Random Hacks of Kindness event in Southampton, UK.

The best way to find out what this project is all about is to look at the presentation that we gave for judging at the #RHoKSoton event - that can be found on Google Docs

The project's code is split into two parts: WaterMe (this repository) which provides the user interface and API, and WaterMeFeeder (repository here) which provides the 'backend' for downloading data, processing it and adding it to a database.

The original problem specification is available here, and the RHoK solutions page is available here.

For more details please see the README files in both repositories.
